Ejercicio 1a. Practica I

A partir de los siguiente valores que se proporciona: true, 5, false, "hola", "adios", 2

a) Crear y asignar variables para cada valor
b) Determinar cual de los dos elementos de texto es mayor
c) Utilizando exclusivamente los dos valores booleanos, determinar los operadores necesarios para obtener un resultado true y otro resultado false
d) Determinar el resultado de las cinco operaciones matemáticas realizadas con los dos elementos numéricos
